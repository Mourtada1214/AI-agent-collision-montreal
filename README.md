# 🤖 Agent IA pour l’analyse des collisions routières à Montréal

## 📌 Description

Ce projet vise à développer une plateforme d’agents d’intelligence artificielle capable d’analyser automatiquement les données de collisions routières à Montréal et de générer des recommandations pour la prise de décision.

## 🎯 Objectif

Automatiser l’analyse descriptive et exploratoire des collisions impliquant les piétons et cyclistes à Montréal.

## 🧠 Approche

- Agents IA spécialisés
- Analyse de données avec Python
- Génération automatique de rapports
- Interface utilisateur simple
